航空航天领域的期刊数量颇为可观，但当前多数“汇总”类文章常将 JIF、CiteScore 及行业杂志数据混为一谈，虽看似排行榜，实则评价口径各异，缺乏参考价值。本文依据**档次**与**领域**对相关期刊进行系统梳理，逐刊展开介绍，旨在帮助读者快速定位适合自身研究成果的投稿目标。

## 档次划分说明

这篇文章把期刊分为四个档次，划分依据是 JCR 分区、CiteScore、学术共同体认可度以及期刊定位，而不是单一数字。

| 档次               | 定义                           | 典型特征                          |
| ---------------- | ---------------------------- | ----------------------------- |
| **第一梯队（顶级刊）**    | JCR Q1，IF > 5，CiteScore > 10 | 综述旗舰刊或综合型高影响力原创刊，审稿严格，国际可见性最强 |
| **第二梯队（主流核心刊）**  | JCR Q1–Q2，IF 2–5             | 传统航空航天核心期刊，方向明确，学术共同体认可度高     |
| **第三梯队（稳健投稿刊）**  | JCR Q2–Q3，IF 1–2             | 区域学会刊、应用导向刊或工程结合型期刊，适合体量适中的工作 |
| **第四梯队（补充与行业刊）** | JCR Q3–Q4 或非研究型出版物           | 行业杂志、区域性小众刊或非标准研究期刊           |

## 第一梯队：顶级刊

### Progress in Aerospace Sciences（PAS）

| 项目 | 内容 |
| --- | --- |
| 出版商 | Elsevier |
| 官网 | [sciencedirect.com/journal/progress-in-aerospace-sciences](https://www.sciencedirect.com/journal/progress-in-aerospace-sciences) |
| JIF | 16.2 |
| CiteScore | 26.7 |
| 出版类型 | Invitation only review journal |

**定位：** 航空航天领域最高级别的综述期刊，不接受自由投稿，所有文章均为邀稿综述。

**适合什么：** 领域全景回顾、前沿总结、系统性综述。如果你受邀撰写大综述，这就是第一选择。

**不适合什么：** 常规原创研究论文——它根本不是原创研究的投稿入口。

---

### Aerospace Science and Technology（AST）

| 项目 | 内容 |
| --- | --- |
| 出版商 | Elsevier |
| 官网 | [sciencedirect.com/journal/aerospace-science-and-technology](https://www.sciencedirect.com/journal/aerospace-science-and-technology) |
| JIF | 5.8 |
| CiteScore | 10.3 |
| 出版类型 | 原创研究 + 综述 |

**定位：** 航空航天综合型原创研究的主力目标刊，覆盖气动、结构、推进、总体设计、方法、实验等方向。

**适合什么：** 大多数航空航天原创工作都会认真考虑 AST 作为首选。它的覆盖面广，审稿体系成熟，是综合航空航天论文的核心候选。

**不适合什么：** 纯工程应用案例——这类稿件更适合工程导向型期刊。

---

### Chinese Journal of Aeronautics（CJA）

| 项目 | 内容 |
| --- | --- |
| 出版商 | Elsevier（中国航空学会主办） |
| 官网 | [sciencedirect.com/journal/chinese-journal-of-aeronautics](https://www.sciencedirect.com/journal/chinese-journal-of-aeronautics) |
| JIF | 5.7 |
| CiteScore | 11.8 |
| 出版类型 | 开放获取，原创研究 + 综述 |

**定位：** 综合型航空航天原创研究期刊，国际可见性已非常强，对国内作者投稿更友好。

**适合什么：** 综合型航空航天原创论文，尤其是希望获得较快审稿周期和较高国际曝光的稿件。CiteScore 已超过 AST，说明其引用影响力在快速上升。

**不适合什么：** 纯区域性或低体量的工程报告。

## 第二梯队：主流核心刊

### AIAA Journal

| 项目 | 内容 |
| --- | --- |
| 出版商 | AIAA（美国航空航天学会） |
| 官网 | [aiaa.org/publications/journals/Journal-Scopes-and-Content/](https://aiaa.org/publications/journals/Journal-Scopes-and-Content/) |
| JIF | 2.8（2024） |
| 出版类型 | 原创研究 |

**定位：** AIAA 旗下的综合型学术期刊，偏基础理论和数值/实验方法，是 aeronautics 和 astronautics 的通用学术平台。

**适合什么：** 基础理论推导、数值方法开发、实验方法创新、通用气动/航天问题。如果你的工作偏"方法"而非"工程实现"，AIAA Journal 比纯工程刊更对口。

**不适合什么：** 纯工程案例、系统设计报告。

### Journal of Aircraft

| 项目 | 内容 |
| --- | --- |
| 出版商 | AIAA |
| 官网 | [aiaa.org/publications/journals/Journal-Scopes-and-Content/](https://aiaa.org/publications/journals/Journal-Scopes-and-Content/) |
| JIF | 2.1（2024） |
| 出版类型 | 原创研究 |

**定位：** 飞机方向的核心专业刊，专注飞机设计、飞行试验、飞行器系统和飞机工程。

**适合什么：** 论文研究对象明确是"飞机"时——总体设计、气动布局、飞行试验数据、结构设计、飞机系统工程。比综合刊更容易找到真正理解飞机问题的审稿人。

**不适合什么：** 纯理论方法（投 AIAA Journal）、纯控制问题（投 JGCD）、非飞机类航天器。

### Journal of Guidance, Control, and Dynamics（JGCD）

| 项目 | 内容 |
| --- | --- |
| 出版商 | AIAA |
| 官网 | [aiaa.org/publications/journals/Journal-Scopes-and-Content/](https://aiaa.org/publications/journals/Journal-Scopes-and-Content/) |
| JIF | 2.8（2024） |
| 出版类型 | 原创研究 |

**定位：** 制导、导航、控制与飞行动力学领域的核心刊物，是 GN&C 方向最明确的学术阵地。

**适合什么：** 如果论文的学术中心就在导航、制导、控制、飞行动力学、状态估计或轨迹优化本身，JGCD 应该是第一梯队选择。这个领域的论文投 JGCD 比投综合刊更容易获得专业审稿和引用。

**不适合什么：** 控制只是整篇论文一个子模块的综合型论文——这类更适合 AST、CJA 等综合刊。

### Journal of the American Helicopter Society（JAHS）

| 项目 | 内容 |
| --- | --- |
| 出版商 | AHS International（现已更名为 VFS——Vertical Flight Society） |
| 官网 | [vtol.org/benefits-and-resources/news-and-publications/journal-of-the-american-helicopter-society](https://vtol.org/benefits-and-resources/news-and-publications/journal-of-the-american-helicopter-society) |
| 出版类型 | 原创研究 |

**定位：** Vertical flight 领域最对口的专业期刊，dedicated to vertical flight technology。

**适合什么：** 直升机、旋翼、tiltrotor、advanced rotorcraft、eVTOL。这类题目投 JAHS 比拿去综合刊硬碰硬更容易获得精准审稿和目标读者。

**不适合什么：** 固定翼飞机、纯航天、非旋翼类飞行器。

### Aerospace（MDPI）

| 项目 | 内容 |
| --- | --- |
| 出版商 | MDPI |
| 官网 | [mdpi.com/journal/aerospace](https://www.mdpi.com/journal/aerospace) |
| JIF | 2.2（2024） |
| JCR 分区 | Q2 |
| 出版类型 | 开放获取（OA） |

**定位：** OA 综合航空航天期刊，范围很宽，流程相对友好。

**不适合什么：** 不建议当作"投不上别的再试试"的默认兜底——稿件质量分层明显，审稿标准在逐步收紧。

| 项目 | 内容 |
| --- | --- |
| 出版商 | Cambridge University Press（英国皇家航空学会 RAeS 主办） |
| 官网 | [cambridge.org/core/journals/aeronautical-journal](https://www.cambridge.org/core/journals/aeronautical-journal/information/about-this-journal) |

## 选刊流程建议

与其盯着数字排名，不如按以下顺序判断：

1. **确定领域**：你的论文核心问题属于哪个方向——综合航空航天、飞机设计、GN&C、旋翼、MAV、可持续航空，还是交叉领域？
2. **确定期刊类型**：综述刊、综合原创刊、专业方向刊、工程应用刊，还是行业出版物？
3. **查看官方指标**：以官网公开的 JIF / CiteScore 为准，不同数据库的指标不要混着比。
4. **检查投稿条件**：OA 或非 OA、APC、审稿周期、投稿系统稳定性。
5. **辅助参考分区**：JCR 分区、中科院分区作为辅助判断，不要当作唯一标准。

一句话总结：

- 写综述 → 先看 **PAS**
- 写综合航空航天原创论文 → 先看 **AST**、**CJA**、**AIAA Journal**
- 写飞机设计和飞机系统 → 优先看 **Journal of Aircraft**
- 写 GN&C 或飞行动力学 → 优先看 **JGCD**
- 写旋翼、VTOL、eVTOL → 不要忘了 **JAHS**
- 写可持续航空、运营、政策交叉 → 看 **IJSA**、**IJAAA**、**Aviation**
